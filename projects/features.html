Astronomical Spectroscopy

Using Spectroscopy to discover our Universe. A NASSP Honours Course

Modeling Spectral Features Project

Due: 1 June 2017

Type: Groups of up to 5 people

Description: The goal of this project is to creata model for the following spectra:

  • A0V star
  • F0V star
  • G0V star
Details of the model are left up to each group, but for the highest mark, it will require modeling all of the spectral features observed in the spectra. For more information about the spectra, see the Pickles Atlas. The models are not flux or luminosity calibrated and have been normalized to an arbritrary value. The temperatures, sizes, and properties of the stars can be looked up based on their spectral type.

Submission: Each group should create a directory within the projects/ directory named features_[groupname]. A report which should include the names of the members of the group, a description of the work (maximum 2000 words), and the final results of the models. Any code created should be included in this directory
