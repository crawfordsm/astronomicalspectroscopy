Reducing SALT Observations

Due:11 Nov 2016

Description: The goal of this project is to reduce and analyze a set of SALT observations. The data set can be downloaded from here . The basic processing has already been taken care of by SALT, only the spectroscopic reduction steps (wavelength calibration, sky subtraction, and extraction) need to be carried out.

Once the data have been reduced, main features and lines should be identified in the spectra, and the feature should be classified. For each of the lines, measure the wavelength center and total flux by fitting a Guassian to the line. From the wavelength center, estimate the redshift for the object. All values should be reported with uncertainties.

PySALT can be installed using anaconda and a PySALT tutorial is also available.

Submission: An ipython notebook or pdf report should be submitted with presenting your work and results. The final results should include the steps taken to reduce the spectra, the results, and how those results were derived. Please include the final spectra and any code written for the project.
